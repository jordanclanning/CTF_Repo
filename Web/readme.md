## Recommended Tools
- **Burp Suite** — intercepting and manipulating HTTP traffic
- **ffuf / gobuster** — directory and parameter fuzzing
- **curl** — quick HTTP requests from the command line
- **browser DevTools** — inspecting requests, cookies, and source
