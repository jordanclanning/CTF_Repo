# Reverse Engineering

Binary reverse engineering challenges organized by operating system and month. Use debuggers, decompilers, and static analysis tools to analyze binaries and recover hidden flags.

---

## Contents

- [Linux](./Linux/) — ELF binary challenges
- [Windows](./Windows/) — PE binary challenges

---

## Recommended Tools

### Linux
- **GDB / pwndbg** — runtime debugging
- **Ghidra** — static decompilation
- **strings / objdump** — quick static analysis
- **ltrace / strace** — library and syscall tracing

### Windows
- **x64dbg / x32dbg** — runtime debugging
- **Ghidra / IDA Free** — static decompilation
- **PEStudio / PE-bear** — PE header analysis
- **Detect-It-Easy** — quick static analysis
